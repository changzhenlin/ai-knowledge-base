# Java RAG / Agent 面试题（初级） - 2026-03-23

## 题目
什么是 Agent？它和普通聊天问答最大的区别是什么？

## 详解答案
普通聊天问答更像“你问我答”，核心是生成文本；Agent 则更像“你给目标，我去执行”。它不仅能回答，还能调用工具、查询数据、执行多步任务。最大的区别是 Agent 具备行动能力和任务编排能力，而不只是语言输出能力。比如查询天气、调用接口、写数据库、发通知，这些都更接近 Agent 的范畴。

## 面试官真正想听什么
- 强调 Agent 有工具调用能力
- 区分问答与执行任务
- 能举一个业务场景例子
