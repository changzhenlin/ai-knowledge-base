# Java RAG / Agent 面试题（高级） - 2026-03-24

## 题目
Java 落地 RAG 时，怎么控制幻觉（hallucination）问题？

## 详解答案
控制幻觉不能只靠 prompt 里一句“请基于资料回答”。更稳的方法是：第一，检索结果必须带来源；第二，生成前明确要求引用证据；第三，当召回置信度低时直接拒答或降级；第四，对敏感领域增加规则校验，比如金额、日期、条款编号。工程上我会把“检索质量”和“生成质量”分开评估，因为很多看起来像模型幻觉的问题，其实根源是召回阶段就错了。

## 面试官真正想听什么
- 幻觉治理要从检索和生成两层同时做
- 低置信度场景要敢于拒答
- 带来源和规则校验很关键
